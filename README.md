# BTS (Pty) Ltd - HR Management System

A desktop applictaion built with **VB.NET** designed to help HR departments manage employee records and system access. This project demonstrates secure database integration using Microsoft Access.

### Key Features
*  **Role Based Login:** Dedicated login module for HR Administrators ( 'HR USER LOGIN.vb' ).
*  **HR Registration Portal:** Allows new HR users to register themselves in the system ( 'HR USER REGISTRATION.vb' ).
*  **Employee Management:** HR can securely add, view and manage employee profiles.
*  **Database Integration:** All user credentials and records are stored in a local **Microsoft Access Database (.mdb)**

### Technology Stack
* **Language:** VB.NET (Visual Basic)
* **Framework:** Windows Forms (.NET Framework)
* **Database:** Microsoft Access ('.mdb')
* **IDE:** Visual Studio

### How to View to Logic
To see the code behind the forms, look at the '.vb' files in this repository:
* **Authentication Logic:** See 'HR USER LOGIN.vb' to see how the system verifies crdentials.
* **Resistration Logic:** See 'HR USER REGISTRATION.vb' for the SQL 'INSERT' commands
